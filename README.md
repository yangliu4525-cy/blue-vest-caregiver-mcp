# 蓝马甲陪护者护航 MCP

> 2026 小X宝开源医疗社区黑客松参赛作品

## 一句话

志愿者粘贴群聊内容 → 输出接话卡 + 行动建议（非诊疗）

## 合规声明

- 不构成诊疗/心理咨询/治疗建议
- 不使用真实患者数据
- 危机拨打 120 或 400-161-9995

## MCP 工具（8个）

| 工具 | 作用 |
|------|------|
| classify_caregiver_moment | 情境识别 |
| generate_peer_response_card | 接话卡 |
| suggest_next_steps | 行动建议 |
| caregiver_load_screening | 陪护者负荷快筛 |
| hospice_resource_checklist | 安宁资源核对清单 |
| route_community_assistant | 社区助手导航 |
| care_handoff_brief | 志愿者交接摘要 |
| full_volunteer_workflow | 一键完整流程（演示推荐） |

## 本地运行

pip install -r requirements.txt
python app.py
